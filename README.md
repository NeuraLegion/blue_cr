# blue_cr

Bluetooth interface using crystal and bluez

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  blue_cr:
    github: NeuraLegion/blue_cr
```

## Usage

```crystal
require "blue_cr"
```

Right now it's pretty much "look at specs"
